Project Overview

CPT208 - Relation Between Generations | Group B4-1

Problem

Stories fade after one conversation

Family memories attached to physical keepsakes are rarely preserved in a format younger users revisit.

Solution

NFC + 3D + messaging loop

Tap a keepsake, unlock the memory visually, and keep the story alive through two-way replies.

Target Users

Three-generation storytelling

  • Grandparents preserving stories
  • Grandchildren learning interactively
  • Parents bridging communication
Deliverables

Submission-ready package

Questionnaire + 50 samples Data statistics dashboard Design process portfolio Timeline page
